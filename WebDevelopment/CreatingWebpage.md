# Creating a Simple Web Page Tutorial

You are a web developer who has been approached by the fan club for IBM Founder, Thomas J. Watson Sr., to create a webpage for them.

## Objectives

- Create a new HTML document
- Specify the DOCTYPE
- Add the head and body elements
- Add a title
- Add a heading
- Add a paragraph
- Add an image
- Create a list of items
- Add a table of data
- Add links to other pages
- Add feedback form to collect user input
